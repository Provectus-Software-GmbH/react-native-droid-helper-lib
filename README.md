# react-native-droid-helper-lib

React Native droid helper library

## Installation

```sh
npm install react-native-droid-helper-lib
```

## Usage


```js
import { multiply } from 'react-native-droid-helper-lib';

// ...

const result = await multiply(3, 7);
```
